# HS Code Reference - Harmonized System Structure

## Purpose
This document provides an overview of the Harmonized System (HS) code structure as maintained by the World Customs Organization (WCO). The HS is a standardized international commodity classification system used for customs tariff classification and trade statistics.

---

## 1. HS Code Hierarchy Overview

### 1.1 Structure Overview

The HS code is a hierarchical, numerical classification system comprising:

| Level | Digits | Name | Description |
|-------|--------|------|-------------|
| Chapter | 2 | HS-2 | Broadest category (01-99) |
| Heading | 4 | HS-4 | Main product grouping |
| Subheading | 6 | HS-6 | Detailed product category |
| National Tariff Line | 8-10 | HS-8/10 | Country-specific extension |

## 3. Heading and Subheading Levels

### 3.1 Heading Structure (4-digit)

Example - Chapter 84 (Nuclear Reactors):
```
84.01 - Nuclear reactors; fuel elements
84.13 - Pumps for liquids
84.21 - Centrifuges
84.24 - Mechanical appliances for projecting liquids
84.31 - Machinery for preparing food or drink
```

### 3.2 Subheading Structure (6-digit)

Example - Heading 84.13 (Pumps for liquids):
```
8401.31 - Pumps fitted or designed to be fitted with a measuring device
          (other than pumps for dispensing fuel or lubricants)
8401.32 - Other pumps for liquids
8401.33 - Pumps for dispensing fuel or lubricants
```

### 3.3 National Tariff Line (8-10 digit)

Countries extend the 6-digit international system to 8 or 10 digits for national tariff purposes:
```
8401.31.00 - United States
8401.31.10 - European Union
8401.31.90 - Other countries
```

## 4. General Interpretation Rules

The HS is interpreted using six General Interpretative Rules (GIRs) that determine classification:

### 4.1 Rule 1 - Primary Rule
Classification is determined according to the terms of the headings and any relative section or chapter notes. If these do not require otherwise, the hierarchy determines classification.

### 4.2 Rule 2 - Incomplete Sets
(a) Reference to a material or substance includes references to that material or substance mixed or combined with other materials.
(b) Goods made of a mixture of materials are classified as if they consisted of the material which gives them their essential character.

### 4.3 Rule 3 - Mixtures and Compounds
When goods cannot be classified by Rules 1 or 2, they are classified under the heading which occurs last in numerical order among those equally applicable.

### 4.4 Rule 4 - Analogy
Goods which cannot be classified by the above rules are classified under the heading appropriate to the goods to which they are most akin.

### 4.5 Rule 5 - Components
(a) Cases, boxes, or similar containers are classified with the goods they are designed to hold.
(b) Packing materials and containers are classified with the goods they pack.

### 4.6 Rule 6 - Subheadings
Classification within subheadings is determined by the terms of those subheadings and related notes, applying rule 1 through 5 by analogy.

---

## 5. Classification Principles

### 5.1 Essential Character Test
For goods made of mixed materials or consisting of multiple components, classification is determined by the material or component which gives the goods their essential character.

### 5.2 Compound Goods
Goods consisting of multiple items packaged together for retail sale are classified as a single unit if they form a composite good.

### 5.3 Parts and Accessories
(a) Parts suitable for use solely or principally with a particular machine are classified with that machine.
(b) Parts used with multiple machines are classified under the generic part classification.

## 7. Classification Decision Process

### 7.1 Step-by-Step Classification

1. **Identify the Chapter:** Determine which chapter applies to the goods
2. **Identify the Heading:** Within the chapter, find the appropriate heading
3. **Identify the Subheading:** Within the heading, determine the applicable subheading
4. **Verify with Notes:** Check section, chapter, and heading notes for additional guidance
5. **Apply GIRs:** Use General Interpretative Rules when classification is not clear

### 7.2 Classification Evidence

The following are considered for classification decisions:
- Physical characteristics of the goods
- Intended use or function
- Design and construction
- Commercial designation
- Technical specifications

## 8. HS Code Structure Summary Table

```
Format: XX.XX.XX.XX
        │  │  │  │
        │  │  │  └── National Tariff Line (optional, 2 digits)
        │  │  └───── Subheading (2 digits)
        │  └────────── Heading (2 digits)
        └───────────── Chapter (2 digits)
```

**Example:**
HS Code: 8471.30.00
- 84 = Chapter (Machinery and boilers)
- 71 = Heading (Automatic data processing machines)
- 30 = Subheading (Portable digital automatic data processing machines)
- 00 = National Tariff Line (unspecified)
