### 2.1 Data Flow Steps

| Step | Component | Action | Privacy Control |
|------|-----------|--------|-----------------|
| 1 | Synthetic Generator | Creates fake declarations, payments, items | No real data |
| 2 | ETL - Validation | Validates schema, checks for PII | Blocks prohibited types |
| 3 | ETL - Transform | Normalizes HS codes, converts currency | No PII transformation |
| 4 | ETL - Load | Loads to staging then curated layers | Write to read-only tables |
| 5 | Materialized Views | Pre-computes aggregations | GROUP BY with threshold |
| 6 | API Endpoint | Serves GET requests | Aggregated JSON only |
| 7 | Dashboard | Displays visualizations | No drill-down enabled |
